# Sandbox Environment Policies

- **Rate Limit:** 1000 requests per hour per API key.
- **Data Reset:** All test data will be wiped every Monday at 00:00 UTC.
- **Data Privacy:** No real patient data should be used in the sandbox.
- **Support:** For issues, contact the support team listed in SUPPORT_CONTACTS.md.
- **Test Credentials:** Provided upon request for integration testing.
- **Webhook Testing:** Use https://webhook.site or your own endpoint for webhook callbacks.
