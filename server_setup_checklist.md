# DHIS2 Server Setup & Import Checklist

## Before Server Setup
- [ ] Confirm all documentation and metadata files are finalized
- [ ] Share necessary files with team and e-Notification team
- [ ] Confirm API authentication details and test credentials
- [ ] Review DHIS2 user roles and permissions for new program
- [ ] Plan test data submission with e-Notification team

## On Server Setup Day (Monday)
- [ ] Deploy DHIS2 version 2.37.10 (if not already running)
- [ ] Import EN_data_elements.json and EN_death_notification_program.json
- [ ] Assign correct organisation units to the new program
- [ ] Configure user roles/permissions for program officers and technical staff
- [ ] Set up API authentication (OAuth2/Basic Auth)
- [ ] Test API endpoints with sample payloads
- [ ] Validate that data appears correctly in DHIS2
- [ ] Coordinate with e-Notification team for live test submission
- [ ] Document any issues or next steps

---

Keep this checklist updated as you proceed. Add/remove steps as needed for your environment.
