# Testing Instructions

## How to Run Tests
- Use your preferred testing framework (e.g., pytest, Mocha, JUnit).
- Run tests with `npm test`, `pytest`, or equivalent command.

## Tools
- Postman or curl for manual API testing
- Automated test scripts (add to SCRIPTS/UTILS)

## Sample Test Cases
- Test authentication (OAuth 2.0 and Basic Auth)
- Test POST, GET, PUT, PATCH, DELETE endpoints
- Test error handling and validation
- Test webhook callback registration
